Generate tailwind.config.js from your design system colors.

## Install

```bash
npm install -g ai-tailwind
```

## Usage

```bash
npx ai-tailwind "#FF4500" "#1A1A2E"
npx ai-tailwind "#3B82F6" "#10B981" "#F59E0B" -o tailwind.config.js
```

```bash
export OPENAI_API_KEY=sk-...
```
